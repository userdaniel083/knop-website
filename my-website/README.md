# My Website

This project is a simple HTML, CSS, and JavaScript website that includes a round button. When clicked, the button triggers a JavaScript animation.

## Project Structure

```
my-website
├── src
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── main.js
│   └── index.html
└── README.md
```

## Getting Started

To run the website, open the `index.html` file in your web browser. 

## Features

- A round button that triggers an animation when clicked.
- Custom styles defined in the `styles.css` file.

## Animation

The animation is handled by JavaScript in the `main.js` file. You can modify the animation effects by editing the JavaScript code.
